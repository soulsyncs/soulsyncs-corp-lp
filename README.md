## ✅ 実装済み機能

### デザインシステム
- **カラーパレット**: 濃紺 `#1E2761` × ティール `#028090` の "Humanity in Logic" 配色
- **タイポグラフィ**: Noto Sans JP + Inter の組み合わせ、可変フォントサイズ
- **余白設計**: CSS変数によるモジュラースケール（xs〜2xl）
- **グリッド**: 12カラムベース、意図的な非対称グリッド崩し

### セクション構成
1. **HERO** — フルスクリーン大ボールドタイポグラフィ + パーティクルCanvas
2. **PHILOSOPHY (01)** — 思想・3バリュー（透明性・共創・進化）
3. **PROCESS (02)** — Black Box → Crystal Clear スクロール連動ジャーニー
4. **SERVICES (03)** — 3サービスカード（業務デジタル化 / システム開発 / IT内製化）
5. **PROOF (04)** — 数字実績 + 顧客の声3件
6. **CEO MESSAGE (05)** — 菊地氏メッセージ + 手書きシグネチャーSVGアニメーション
7. **RECRUIT (06)** — 採用3ポジション
8. **CONTACT (07)** — お問い合わせフォーム（API連携済み）

### アニメーション・インタラクション
- **カスタムカーソル** — マウス追従 + ホバー時拡張
- **ローダー画面** — ブランドロゴ + プログレスバー
- **スクロールリビール** — IntersectionObserver による滑らかな出現
- **ヒーローパーティクル** — Canvas上の粒子 + ネットワーク線
- **カウンターアニメーション** — イージングつき数値カウントアップ
- **シグネチャードロー** — SVGパスドロー（CEO手書き署名）
- **カードチルト** — マウス追従3D回転
- **マグネットボタン** — マウスに引き寄せられるCTAボタン
- **ページ進捗バー** — 画面上部スクロール進捗インジケーター
- **グリッチエフェクト** — ヒーロー見出しへのランダムな揺らぎ
- **ジャーニー進捗バー** — スクロール連動のBlack Box→Crystal Clear進捗
- **パラックス** — ヒーローセクションの背景要素視差

### レスポンシブ対応
- モバイル（480px以下）・タブレット（768px）・デスクトップ完全対応
- タッチデバイスではカスタムカーソルを非表示

## 📡 APIエンドポイント・パス

| パス | メソッド | 説明 |
|------|----------|------|
| `/` | GET | コーポレートLP（index.html） |
| `tables/contact_requests` | POST | お問い合わせ送信 |
| `tables/contact_requests` | GET | お問い合わせ一覧（管理用） |

### フォームパラメータ
```json
{
  "company": "株式会社〇〇",
  "name": "山田 太郎",
  "email": "taro@example.com",
  "topics": ["dx", "system"],
  "message": "課題のテキスト..."
}
```

---

## 🗃️ データモデル

### `contact_requests` テーブル
| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | text | 自動生成UUID |
| company | text | 会社名 |
| name | text | 担当者名 |
| email | text | メールアドレス |
| topics | array | 相談内容（dx/system/it/other） |
| message | rich_text | 状況・課題の詳細 |
| created_at | datetime | 送信日時（自動） |
